Application Summary

This application was developed for use at my place of work. They were using Excel spreadsheets to keep track of credit card expenses, and I proposed this project as a way to make that process more efficient. Not only did I get class credit, but I got to do the work for class mostly at work, getting paid for it.

Code Samples

Database connection for project to local SQL database:

                    
                        public static class ConnectionDB
                        {

                            public static SqlConnection GetConnection()
                            {
                                string connectionString =
                                "Data Source=(LocalDB)\\v11.0;AttachDbFilename=|DataDirectory|" +
                                "\\ExpenseItems.mdf;Integrated Security=True";
                                SqlConnection connection = new SqlConnection(connectionString);
                                return connection;
                            }
                        }
